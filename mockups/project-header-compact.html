Project Header — Compact Layout Ideas
Today's project header is a 2-line "← Back to projects" link + a name row with × close + a meta row + a client line. That's roughly ~92 px tall and pushes everything below it down. Each idea below combines the back/× into one affordance AND restructures the info layout. Sidebar bodies are clipped to the same fixed height so you can read off "more rows visible = more compact header."
Baseline · Current~92 px
Today's pattern. Two affordances do the same thing (← Back link + × button), spread across 4 lines (back / name / meta / client).

Quick recommendation
Idea 2 (Workspace switcher) is my top pick — caret-dropdown is the most familiar pattern for "I'm in X, take me out / move me to Y," and it doubles as a project switcher (huge time-save vs. having to back out and pick another). Idea 1 (Breadcrumb) is the most space-efficient if you only need exit (no switch). Idea 4 (Bordered card) wins on "you are inside this thing" feel. Pick one — happy to refine into the final shipping pattern.
