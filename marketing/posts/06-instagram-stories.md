# Instagram Stories — first set (3 sets × 5–7 frames each)

> Stories are vertical 9:16, swipe-through. Each "set" tells a 5–7 frame mini-story.
> Use stickers (poll, question, slider, link) to drive engagement — Instagram favours stories with stickers in its algorithm.
> The "Link" sticker should be on the final frame, pointing to procabinet.app (with a UTM: `?utm_source=instagram&utm_medium=story&utm_campaign=launch`).
> Save the best set as a **Highlight** on your profile so new visitors see it.

---

## SET A — "How I built this" (founder origin, 6 frames)

**Why:** trades audiences trust the person before the product. This set introduces you, your shop, and what you built — in a casual, talking-to-camera way. Run as your **first ever story** the day you launch.

| # | Visual | Top-of-frame text | Bottom-of-frame text | Sticker |
|---|---|---|---|---|
| 1 | Selfie / talking-head, workshop or laptop in background | **"Hi — I'm Adam"** | "Just launched something I've been building 👇" | None |
| 2 | Photo of a cabinet you made (or your old workshop) | **"I ran a 5-man cabinet shop for 14 years"** | "Held the whole thing together with spreadsheets" | None |
| 3 | Screenshot of an Excel quote sheet (use a real-looking one — anonymous data) | **"This was the bottleneck I never quite fixed"** | "Quoting took hours. I was losing jobs to faster shops." | Reaction sticker (😩 or "ME") |
| 4 | Talking-head / hands typing | **"So I built my own"** | "Quote time went from hours → minutes. Could finally hand it to an assistant." | None |
| 5 | Static screenshot of the ProCabinet.App Dashboard | **"Now it's a tool anyone can use"** | "Quote, cut list, schedule, stock — one place. Free to try." | Quiz sticker: *"What's your quoting bottleneck?"* options: Time / Maths / Format |
| 6 | Logo on amber-on-black background, big | **"ProCabinet.App"** | "Free plan + $299 founders' lifetime (50 spots)" | **Link sticker → procabinet.app** + countdown sticker for the lifetime cap |

**Recording tips:**
- Hold phone vertically, eye level
- Natural light from a window, not overhead
- Frame 1 + 4: talk for 5–10 seconds, no script — just say it
- Use IG's built-in "Type" tool for the text, not a separate graphic editor
- Save each frame for &gt;3 seconds (people skip fast)

## Story stickers cheat-sheet

| Sticker | When to use |
|---|---|
| **Link** | Always on final frame. The whole point. |
| **Poll** | Anywhere you want a yes/no engagement bump |
| **Quiz** | Ask the audience to commit to an answer (high engagement) |
| **Question** | Frame 5 of any set — "What feature should I build next?" |
| **Slider (emoji)** | Reaction to a feature reveal — fun, low-friction |
| **Countdown** | Counting down to the Lifetime cap closing — urgency without pressure |
| **@Mention** | Tag a customer / creator who's using the app |

## Daily story cadence (launch month)

- **Day 1:** Set A (introduction)
- **Day 2:** Behind-the-scenes (you replying to comments / packing creator merch / shop visit)
- **Day 3:** Set B (5 things)
- **Day 5:** Repost a customer's screenshot
- **Day 7:** Founders' Lifetime countdown (3 frames: the cap number / why / link)
- **Day 10:** Set C (before/after)
- **Day 14:** Q&amp;A — paste 3 DM questions you've received and answer on camera

After day 14, settle into 1 story per day minimum — even just a screenshot, a customer kind word, a behind-the-scenes shot.
