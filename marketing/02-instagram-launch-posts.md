# Instagram launch posts — 3 sets

> Each set = one Instagram caption ready to ship + the image / video direction it needs.
> First 9 posts on a brand-new account define the grid — pick one of these three sets and run **all three posts in that set** in the first week, in the order shown, so the grid tells a story.
> Use the **square logo** (`assets/logo-square.svg`) as the profile picture.
